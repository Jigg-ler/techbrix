PC BUILDER

What are we?

Because of the COVID-19 outbreak, there is an increase usage of computers and cloud computing applications since people, especially employees, are working at home to prevent contagion and heavily rely on computers (Z. R. Alashhab et al., 2021). To maximize this trend, this web application has been developed. PC Builder is a web application that acts as a guide for computer building and a store that sells computer parts and systems. This web application will provide a list of different computer parts that are currently available with its detailed information. There will be a compatibility check feature that will make sure that the parts selected by the client will work together. Computer packages with different price ranges will also be offered for the clients who do not have prior knowledge in computer building. An e-commerce system will be utilized for this project for handling transactions involve within the web application.
